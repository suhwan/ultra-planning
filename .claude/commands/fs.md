# /fs

`/fresh-start`의 단축 명령어.

## 인자: $ARGUMENTS

## 실행

이 명령어는 `/fresh-start`와 동일합니다.
`/fresh-start $ARGUMENTS` 를 실행하세요.

## 사용 예시

```bash
/fs              # = /fresh-start
/fs continue     # = /fresh-start continue
/fs phase 2      # = /fresh-start phase 2
/fs plan 01-02   # = /fresh-start plan 01-02
```
